Visualisasi Data dengan Principal Component Analysis dan Linear Discriminant Analysis

Dataset kami memiliki 13 kolom. Bagaimana cara visualisasi semua kolom tersebut sekaligus? Mata kita melihat dunia secara 3D. Akibatnya, plotting / visualisasi data dengan dimensi > 4 itu sulit. Salah satu metode yang dapat kita lakukan adalah dimensionality reduction, atau reduksi dimensi. PCA dan LDA adalah dua teknik statistik yang digunakan untuk mengurangi dimensi data. PCA maupun LDA menggunakan transformasi linear untuk mengubah data dari ruang asli ke ruang baru dengan dimensi yang lebih rendah. Metode tersebut mencari kombinasi linear dari variabel asli yang memungkinkan representasi data yang lebih ringkas.

Klasifikasi dengan K-Nearest Neighbors

K-nearest neighbors (KNN) adalah algoritma machine learning yang bisa digunakan untuk klasifikasi dan regresi. Prinsip dasar dari KNN adalah menentukan kategori atau nilai suatu titik data berdasarkan kedekatan atau kemiripan dengan titik data lainnya dalam dataset. KNeighborsClassifier dari scikit-learn menerima berbagai macam konfigurasi (hyperparameter). Contohnya adalah n_neighbors atau jumlah neighbors / tetangga yang dipertimbangkan, dan metric yang mengontrol bagaimana cara menghitung jarak antara satu data ke data lain.

Membagi dataset menjadi train dan test set

Model yang belajar dengan 100% dari dataset dan divalidasikan dengan dataset yang sama mungkin akan memiliki performa yang baik. Tapi, tujuan kita membuat model adalah untuk memprediksi data yang belum pernah dilihat oleh model tersebut pada saat training (generalisasi). Jadi, untuk pengujian yang lebih akurat, kita bagi dataset menjadi training set dan testing set. Training set digunakan untuk melatih model dan testing set digunakan untuk validasi / test model.

Disini kami gunakan fungsi train_test_split dengan parameter test_size=0.2 yang artinya kita ambil 20% dari dataset untuk test set dan sisanya untuk training set, parameter random_state=37 supaya hasil pembagiannya bisa direproduksi, dan stratify=y yang memastikan bahwa data yang dibagi itu memiliki distribusi proporsi yang sama seperti dataset asli.

KNN dengan Standardized Data

Preprocessing

Feature dengan skala yang lebih besar dapat mendominasi fitur lain dan menyebabkan algoritma belajar dengan bias. Karena range atau skala dari setiap feature yang ada pada dataset berbeda, maka kita standardisasi data terlebih dahulu dengan StandardScaler. Metode ini juga bisa mengurangi dampak outlier.

\(x' = \frac{\text{nilai x} - \text{mean(x)}}{\text{std(x)}}\)

stdscale = StandardScaler().fit(X_train)
X_train_std = stdscale.transform(X_train)
X_test_std = stdscale.transform(X_test)
df_scaled = pd.DataFrame(X_train_std, columns=X_train.columns)

Tuning KNN dengan GridSearchCV

Supaya mendapatkan akurasi yang lebih baik, kita cari hyperparameter dengan menggunakan GridSearchCV.

param_grid merupakan map berupa parameter yang akan dicoba selama grid search.

Mencari Hyperparameter Terbaik

Code 
model = KNeighborsClassifier()

param_grid = {
    'n_neighbors': np.arange(1, 31),
    'weights': ['uniform', 'distance'],
    'metric': ['euclidean', 'manhattan', 'minkowski']
}

grid_search = GridSearchCV(model, param_grid, cv=4, n_jobs=-1, verbose=2)
grid_search.fit(X_train_std, y_train)

print(f"Best parameters: {grid_search.best_params_}")
print(f"Best cross-validation (cv) score: {grid_search.best_score_}")

Tolerant Accuracy

Code 
def tolerant_accuracy(y_true, y_pred, tolerance=1):
    return np.mean(np.abs(y_true - y_pred) <= tolerance)

model = KNeighborsClassifier(n_neighbors=28, metric='manhattan', weights='distance')
model.fit(X_train_std, y_train)

y_pred = model.predict(X_test_std)

conf_mat = confusion_matrix(y_test, y_pred)
disp = ConfusionMatrixDisplay(confusion_matrix=conf_mat,
                              display_labels=model.classes_)
disp.plot()
report = classification_report(y_test, y_pred, zero_division=0.0)
print(report)
display(f"Tolerant accuracy: {tolerant_accuracy(y_test, y_pred) * 100}%")
